# Specification

## Summary
**Goal:** Build KidZone, a colorful interactive app for young children (ages 3–8) featuring learning, games, and drawing activities — no login required.

**Planned changes:**
- Home screen with a large animated title, sun mascot illustration, and three clearly labeled activity buttons (Letters & Numbers, Memory Match, Draw)
- Letters & Numbers section: A–Z and 1–20 card grid; tapping a card shows a full-screen animated pop-up with the letter or number in large colorful text
- Memory Match mini-game: 4×4 grid of face-down animal cards, flip-to-reveal animation, match tracking, congratulatory message on completion, and persistent best-score display ("Best: X moves") backed by the database
- Color & Draw canvas: full-width drawing area, 8-color palette, mouse and touch drawing support, and a Clear button
- Joyful visual theme throughout: bright primary colors, rounded bubbly shapes, chunky rounded sans-serif typography, star/confetti decorative accents, and minimum 48×48px tap targets on all interactive elements

**User-visible outcome:** Children can visit the app immediately without logging in and independently explore letter/number learning cards, play a memory match card game with a saved best score, and draw freely on a canvas — all within a bright, playful, easy-to-navigate interface.
